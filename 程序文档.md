\# 一、项目概述



本项目是一款基于 Android 平台开发的租房信息管理系统，采用 \*\*MVVM 架构 + 前后端分离设计\*\*，实现用户对房源的浏览、发布、收藏、聊天及地图定位等功能。



系统通过 RESTful API 与后端进行数据交互，数据格式统一为 JSON。

\# 三、系统架构设计（MVVM）



\## 架构结构



```

View（UI层）

&#x20;   ↓

ViewModel（逻辑控制层）

&#x20;   ↓

Repository（数据仓库层）

&#x20;   ↓

Service（业务逻辑层）

&#x20;   ↓

ApiClient（网络请求层）

&#x20;   ↓

Backend API（后端服务）

```



\## 各层说明



\* \*\*View层\*\*：负责UI展示（Activity）

\* \*\*ViewModel层\*\*：处理数据逻辑与UI绑定

\* \*\*Repository层\*\*：统一数据来源

\* \*\*Service层\*\*：业务逻辑封装

\* \*\*ApiClient层\*\*：网络请求封装

\* \*\*Backend\*\*：提供REST API
